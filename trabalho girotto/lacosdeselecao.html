Laços de Seleção em C

Em C, os laços de seleção permitem que o programa faça escolhas com base em condições específicas. Os principais laços de seleção são if, else if e else.

if

O laço if é usado para executar um bloco de código se uma condição for verdadeira. Por exemplo:

        
#include <stdio.h>

int main() {
    int idade = 20;
    if (idade >= 18) {
        printf("Você é maior de idade.\n");
    }
    return 0;
}

else if

O laço else if permite adicionar condições adicionais após um if inicial. Ele é usado quando há mais de uma condição que precisa ser verificada. Por exemplo:

        
#include <stdio.h>

int main() {
    int nota = 75;
    if (nota >= 90) {
        printf("Você tirou A.\n");
    } else if (nota >= 80) {
        printf("Você tirou B.\n");
    } else if (nota >= 70) {
        printf("Você tirou C.\n");
    } else {
        printf("Você tirou D ou F.\n");
    }
    return 0;
}

else

O laço else é usado para executar um bloco de código se a condição de um if ou de um else if anterior for falsa. Por exemplo:

        
#include <stdio.h>

int main() {
    int idade = 15;
    if (idade >= 18) {
        printf("Você é maior de idade.\n");
    } else {
        printf("Você é menor de idade.\n");
    }
    return 0;
}

Estes são exemplos básicos de como usar os laços de seleção em C. Eles são fundamentais para tomar decisões dentro de um programa com base em condições específicas.
