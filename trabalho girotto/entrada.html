Entrada e Saída de Dados em C

Em C, a entrada e saída de dados são realizadas principalmente usando as funções scanf e printf.

scanf()

A função scanf é utilizada para receber dados de entrada do usuário. Ela espera uma sequência de especificadores de formato, correspondendo aos tipos de dados das variáveis que receberão os valores. Por exemplo:

        
#include <stdio.h>

int main() {
    int idade;
    printf("Digite sua idade: ");
    scanf("%d", &idade);
    printf("Sua idade é %d anos.\n", idade);
    return 0;
}

printf()

A função printf é usada para exibir dados na tela. Assim como scanf, ela também utiliza especificadores de formato para indicar o tipo de dados que será exibido. Por exemplo:

        
#include <stdio.h>

int main() {
    int num1 = 10, num2 = 20;
    printf("A soma de %d e %d é %d.\n", num1, num2, num1 + num2);
    return 0;
}

Estes são exemplos básicos de como usar scanf e printf para entrada e saída de dados em C. Existem muitas outras opções e especificadores de formato disponíveis, dependendo das necessidades do programa.
